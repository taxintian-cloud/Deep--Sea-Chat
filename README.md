# 🌊 Deep Sea Chat

深海の静寂に、言葉を投げる。  
**Deep Sea Chat** は、海神のような人格を持つAIと対話できる  
フロントエンド完結型のチャットアプリです。

幻想的な深海UIと、詩的な応答を通して  
「癒し」「対話」「没入感」をテーマに制作しました。

---

## ✨ 特徴

- 🐚 **深海×AI人格チャット**
  - 海神をイメージした独自人格ロジック
  - 挨拶・質問・感謝・疲労などを簡易的に意図判定

- 💬 **スレッド型チャット管理**
  - 会話ごとにスレッド作成
  - タイトルの編集・削除が可能
  - `localStorage` による永続化

- 🔮 **真珠オーブのインタラクション**
  - 発話時の発光・波紋エフェクト
  - typing中の淡い光アニメーション

- 🌌 **幻想的なUI/UX**
  - 深海グラデーション背景
  - 浮遊するバブルレイヤー
  - ガラス調（Glassmorphism）デザイン

## 🧠 実装ポイント

- **状態管理**
  - スレッド・メッセージ・現在の会話を `localStorage` で管理
  - 初回アクセス時の自動スレッド生成

- **会話ロジック**
  - 正規表現による簡易Intent判定
  - 人格別の返答プールからランダム応答

- **UX配慮**
  - IME入力中のEnter送信防止
  - フォーカスリング対応
  - アクセシビリティ（視覚的非表示要素）
